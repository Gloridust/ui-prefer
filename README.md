# 软件风格预览工具

**熵析云枢网络科技**出品的 UI 风格预览工具，帮助客户直观地挑选和预览不同的 UI 设计方案。

## ✨ 功能特性

### 🎨 配色系统
- **8 种预设主题色**：蓝色、紫色、粉色、绿色、橙色、红色、青色、黄色
- **自定义颜色选择器**：支持任意颜色输入
- **智能配色生成**：根据主题色自动生成完整的配色方案，包括：
  - 主色及其明暗变体
  - 辅助色和强调色
  - 语义化颜色（成功、警告、错误、信息）

### 🎭 7 种 UI 设计风格

1. **扁平化 (Flat)**
   - 简洁现代的设计语言
   - 清晰的层次结构
   - 适合商务和工具类应用

2. **新拟态 (Neumorphic)**
   - 柔和的 3D 浮雕效果
   - 阴影塑造立体感
   - 现代时尚的设计风格

3. **玻璃态 (Glassmorphic)**
   - 透明模糊的玻璃质感
   - 半透明背景效果
   - 高端精致的视觉体验

4. **iOS 风格**
   - 苹果系统设计语言
   - 圆角和阴影的完美结合
   - 优雅简约的界面

5. **Material Design**
   - 谷歌材质设计规范
   - 纸片层叠的隐喻
   - 清晰的交互反馈

6. **圆角卡片 (Rounded)**
   - 大圆角柔和风格
   - 友好温暖的视觉感受
   - 适合社交和生活类应用

7. **野性主义 (Brutalist)**
   - 粗犷原始的设计风格
   - 强烈的边框和阴影
   - 个性鲜明的视觉冲击

### 📱 响应式预览

- **桌面端视图**：展示完整的桌面应用界面
- **移动端视图**：模拟手机屏幕的显示效果
- 一键切换，实时预览

### 🧩 丰富的组件展示

预览工具包含以下常见组件：

- **导航栏**：带关闭/最小化按钮的顶部栏
- **标签页**：多标签页切换效果
- **按钮**：主要按钮、次要按钮、边框按钮
- **表单输入**：文本框、邮箱输入、多行文本域
- **选择控件**：开关按钮、复选框
- **滑块**：音量控制等滑块组件
- **卡片列表**：网格布局的内容卡片
- **通知提示**：成功、警告、错误、信息提示框

## 🚀 快速开始

### 安装依赖

```bash
yarn install
# 或
npm install
```

### 运行开发服务器

```bash
yarn dev
# 或
npm run dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看效果。

### 构建生产版本

```bash
yarn build
# 或
npm run build
```

### 启动生产服务器

```bash
yarn start
# 或
npm start
```

## 🎯 使用指南

1. **选择主题色**
   - 点击预设颜色快速应用
   - 使用颜色选择器自定义颜色
   - 输入十六进制颜色代码

2. **选择 UI 风格**
   - 浏览 7 种设计风格
   - 点击任意风格卡片切换
   - 右侧实时预览效果

3. **切换视图模式**
   - 点击"桌面端"查看电脑界面
   - 点击"移动端"查看手机界面
   - 预览区域自动适配

4. **查看组件效果**
   - 滚动预览区域查看所有组件
   - 与组件交互测试效果
   - 观察不同风格的差异

## 🎨 配色方案生成逻辑

系统根据选择的主题色，通过 HSL 色彩空间转换，自动生成：

- **主色明暗变体**：亮度 ±15%
- **辅助色**：色相旋转 180°
- **强调色**：色相旋转 30°，饱和度 +10%
- **中性色**：固定的背景、文本、边框色
- **语义色**：成功（绿）、警告（黄）、错误（红）、信息（蓝）
